## Compile from Source
To install **mfg-dl**, follow these steps:
1. Clone the repository:
    ```shell
    git clone https://github.com/Deskilling/mfg-dl.git
    ```
2. Navigate to the project directory:
    ```shell
    cd mfg-dl
    ```
3. Build the project:
    ```shell
    go build .
    ```
